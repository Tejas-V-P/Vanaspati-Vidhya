# Vanaspati Vidya 🌿✨

An interactive, full-stack educational web application dedicated to the botanical science and profound wisdom of Ayurveda. Built with a modern React SPA architecture, custom CSS wizardry, and an Express.js backing API, Vanaspati Vidya bridges ancient Indian Knowledge Systems (IKS) with contemporary web design.

### 🌟 Key Features
*   **Dynamic Plant Database:** Explore 15 sacred medicinal plants (like Tulsi, Ashwagandha, and Arjuna) complete with traditional uses, modern scientific benefits, and unique CSS gradient aesthetic profiles.
*   **Alchemy Cauldron:** A highly interactive "Cauldron" engine that allows users to magically mix up to 3 herbs and discover authentic classical Ayurvedic formulations (like *Immunity Kadha* and *Golden Milk*).
*   **Interactive Knowledge Quiz:** A gamified learning experience designed to test and reinforce user knowledge of ancient balancing (Doshas) and botanical medicine.
*   **Modern Full-Stack Architecture:** 
    *   **Frontend:** Vite, React, React Router (Seamless client-side routing)
    *   **Backend:** Node.js, Express.js REST API
    *   **Styling:** Custom Vanilla CSS with buttery-smooth micro-interactions, scroll reveals, and responsive typography avoiding heavy CSS frameworks.

### 🚀 Get Started Locally
```bash
# Clone the repository
git clone https://github.com/your-username/vanaspati-vidya.git

# Install dependencies
npm install

# Start both the Vite Frontend and Express Backend concurrently
npm run start
```
*Note: Make sure your terminal runs this from the project root!*

### 🛠 Tech Stack
*   **React 18** + **Vite**
*   **React Router DOM**
*   **Node.js** + **Express**
*   **Vanilla CSS3** (Custom Animations & Gradients)
